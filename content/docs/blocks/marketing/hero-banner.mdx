A modern hero banner component with background image, title, description, and customizable call-to-action buttons.

### Usage

```tsx
import { HeroBanner } from "@/components/blocks/hero-banner";

export default function HomePage() {
  return (
    <HeroBanner
      title="Welcome to Our Platform"
      description="Build amazing products with our tools"
      primaryAction={{
        label: "Get Started",
        href: "/signup"
      }}
      secondaryAction={{
        label: "Learn More",
        href: "/about"
      }}
      backgroundImage="/hero-bg.jpg"
    />
  );
}
```

### Props

| Prop | Type | Description |
|------|------|-------------|
| `title` | `string` | Main heading text (required) |
| `description` | `string` | Subheading text (optional) |
| `primaryAction` | `{ label: string, href?: string, onClick?: () => void }` | Primary CTA button (optional) |
| `secondaryAction` | `{ label: string, href?: string, onClick?: () => void }` | Secondary CTA button (optional) |
| `backgroundImage` | `string` | Background image URL (optional) |
| `className` | `string` | Additional CSS classes (optional) |

### Features

- Fully responsive design (mobile, tablet, desktop)
- Background image with dark overlay for text readability
- Customizable actions (links or onClick handlers)
- TypeScript support with full type safety
- Built with shadcn/ui components
- Accessible and semantic HTML
