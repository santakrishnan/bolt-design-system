## Usage

---

The Chart component is built on top of Recharts and provides a simple way to create beautiful, accessible charts.

```tsx
import { ChartContainer, ChartTooltip, ChartTooltipContent } from "@/components/ui/chart"
```

For detailed examples and usage, please refer to the [shadcn/ui Chart documentation](https://ui.shadcn.com/docs/components/chart).
