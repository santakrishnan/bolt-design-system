## Build Your Own

You can build your own blocks using our component library. All components are available at `/docs/components`.

### Example Block Pattern

```tsx
import { Button } from "@/components/ui/button";
import { Card, CardContent, CardHeader, CardTitle } from "@/components/ui/card";

export function MyBlock() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>My Custom Block</CardTitle>
      </CardHeader>
      <CardContent>
        <p>Build amazing blocks with shadcn/ui components!</p>
        <Button className="mt-4">Get Started</Button>
      </CardContent>
    </Card>
  );
}
```
