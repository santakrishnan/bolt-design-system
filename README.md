### Usage Example

To add the `button` component to your project, run the following command:

```bash
pnpm dlx shadcn@latest add @bolt-design/button
```

Once installed, you can import and use the component in your files:

```typescript
import { Button } from "@/components/ui/bolt";

export default function App() {
  return <Button>Click me</Button>;
}
```

**Note:** The import path `@/components/ui/bolt` assumes your project has a path alias configured (common in Next.js and similar frameworks). Adjust the path to match your project's structure if needed.
