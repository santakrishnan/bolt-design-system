## Level Up Animation

When the XP bar reaches 100%, it displays a flashing animation with a "LEVEL UP!" message:

```tsx
<XpBar value={100} />
```

You can customize the level up message:

```tsx
<XpBar value={100} levelUpMessage="MAX LEVEL!" />
```
