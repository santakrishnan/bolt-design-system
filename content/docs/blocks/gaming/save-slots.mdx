## Usage

---

```tsx
import { SaveSlots } from "@/components/ui/8bit/blocks/save-slots"
```

```tsx
<SaveSlots
  slots={[
    {
      id: "1",
      isEmpty: false,
      name: "My Adventure",
      timestamp: "2025-12-01T14:30:00",
      preview: "/path/to/preview.png",
      description: "Level 15 - Crystal Caves",
    },
    {
      id: "2",
      isEmpty: true,
    },
  ]}
  onSlotClick={(slot) => {
    if (slot.isEmpty) {
      console.log("Create new save");
    } else {
      console.log("Load save:", slot.name);
    }
  }}
  layout="vertical"
  title="SAVE FILES"
  showTimestamp={true}
  showPreview={true}
/>
```

## Props

The `SaveSlots` component accepts the following props:

- **slots** (required): Array of `SaveSlot` objects containing save data
- **onSlotClick**: Callback function triggered when a slot is clicked
- **layout**: Choose between `"vertical"` or `"grid"` layout (default: `"vertical"`)
- **maxSlots**: Maximum number of slots to display (default: `10`)
- **title**: Custom title for the save slots card (default: `"SAVE FILES"`)
- **showTimestamp**: Display last save timestamp (default: `true`)
- **showPreview**: Display preview images (default: `true`)
- **className**: Additional CSS classes for styling

### SaveSlot Interface

```typescript
interface SaveSlot {
  id: string;
  isEmpty: boolean;
  name?: string;
  timestamp?: Date | string;
  preview?: string;
  description?: string;
}
```


### Custom Slot Limit

```tsx

  title="QUICK SAVES"
```
